# VBA DataImport モジュール - Excel導入手順

## 前提条件

- Excel 2016以降（Windows）
- 対象ファイル: `【保護解除】耐用年数評価各チェックシート_20260130【委託検査員送付】v1.3.xlsm`

---

## Step 1: 参照設定

**追加の参照設定は不要です。** Mac/Windows両対応のため、外部ライブラリへの依存を排除しています。

---

## Step 2: モジュールのインポート

VBAエディタで以下の4ファイルをインポートします。

1. Excelファイルを開く
2. **Alt+F11**（Macは **Option+F11** または **ツール→マクロ→Visual Basic Editor**）でVBAエディタを開く
3. メニュー **ファイル → ファイルのインポート** を選択
4. 以下のファイルを1つずつインポート:

| 順番 | ファイル | 種類 | 説明 |
|------|---------|------|------|
| 1 | `Dictionary.cls` | クラスモジュール | Dictionary互換クラス（Mac/Windows両対応） |
| 2 | `JsonConverter.bas` | 標準モジュール | JSON解析ライブラリ（Tim Hall, MIT License） |
| 3 | `DataImport.bas` | 標準モジュール | メインモジュール（データ取り込み処理） |
| 4 | `CellAddressCollector.bas` | 標準モジュール | マッピングシート作成＆セルアドレス収集ツール |

インポート後の確認:
- 「クラスモジュール」フォルダに `Dictionary` が追加されていること
- 「標準モジュール」フォルダに3つのモジュールが追加されていること

---

## Step 3: ThisWorkbookにイベントコードを追加

セルアドレス収集ツールがセルクリックを検知するために必要です。

1. VBAエディタの左側「プロジェクトエクスプローラ」で **ThisWorkbook** をダブルクリック
2. 以下のコードを貼り付け:

```vba
Private Sub Workbook_SheetSelectionChange(ByVal Sh As Object, ByVal Target As Range)
    If Sh.Name = "現地調査" Then
        CellAddressCollector.OnCellSelected Target
    End If
End Sub
```

3. **Ctrl+S** で保存（.xlsm形式であることを確認）

---

## Step 4: マッピングシートの作成

現地調査シートのセルアドレスをマッピング定義するシートを作成します。

1. **Alt+F8** でマクロ一覧を開く
2. `CreateMappingSheet` を選択して **実行**
3. 「マッピング」シートが自動生成される

生成されるテーブル:

| テーブル | 内容 | 状態 |
|---------|------|------|
| A: 評価セル | item1-93のb2/cセル | **自動入力済み**（W/Y列） |
| A: 調査方法 | 目視/計測のセル | 要入力（黄色セル） |
| B: グループ有無 | 擁壁/駐車場等の有/無 | 要入力（黄色セル） |
| C: オプション選択 | 構造種別/屋根仕様等 | 要入力（黄色セル） |
| D: メンテナンス | 補修要否/状態 | 要入力（黄色セル） |
| E: カテゴリ調査状況 | 調査実施/不可 | 要入力（黄色セル） |

---

## Step 5: セルアドレスの収集（黄色セルを埋める）

1. **Alt+F8** → `CollectCellAddresses` を実行
2. 自動的に「現地調査」シートに切り替わる
3. **ステータスバー**（画面下部）に指示が表示される:

   ```
   [1/120] 【調査方法/目視】item1 敷地及び地盤: 地盤沈下 ← このセルをクリック
   ```

4. 指示に従って該当セルをクリック
5. 確認ダイアログが表示される:
   - **はい** → 記録して次へ
   - **いいえ** → やり直し（別のセルを選択）
   - **キャンセル** → この項目をスキップ
6. 全項目完了まで繰り返し

### 中断・再開

- **中断**: Alt+F8 → `StopCollecting` を実行
- **再開**: Alt+F8 → `CollectCellAddresses` を再実行（入力済み項目はスキップされる）

### 入力状況の確認

- **Alt+F8** → `ValidateMappingSheet` を実行
- 各テーブルの空欄数が表示される
- 全て0件なら準備完了

---

## Step 6: JSONデータ取り込みボタンの配置（任意）

1. **Alt+F8** → `CreateImportButton` を実行
2. 現地調査シートの左上に「JSONデータ取り込み」ボタンが配置される

---

## Step 7: データ取り込みの実行

### ボタンから実行

現地調査シートの「JSONデータ取り込み」ボタンをクリック

### マクロから実行

1. **Alt+F8** → `ImportJsonData` を実行
2. ファイル選択ダイアログでJSONファイルを選択
3. 確認ダイアログで「はい」
4. 処理が自動実行される（進捗はステータスバーに表示）
5. 完了メッセージが表示される

### 取り込まれるデータ

| 対象 | 内容 |
|-----|------|
| 現地調査シート | 物件情報、評価値（■マーク）、調査方法、グループ有無等 |
| 劣化事象シート | 階層別に自動生成（評価c/b2の不具合画像・テキスト） |
| キープランシート | 階層別に自動生成（図面+ピン+サムネイル+矢印線） |

---

## トラブルシューティング

### 「マッピングシートが見つかりません」エラー

→ Step 4の `CreateMappingSheet` を先に実行してください

### 「型が一致しません」エラー

→ Dictionary.cls がインポートされているか確認（クラスモジュールとして存在すること）

### 画像が挿入されない

→ JSONデータ内の画像がWebP形式の場合、WIA.ImageFileによる変換を試みます。変換に失敗した場合はエラーログに記録されます

### セルアドレス収集でクリックが反応しない

→ Step 3のThisWorkbookイベントコードが正しく追加されているか確認
