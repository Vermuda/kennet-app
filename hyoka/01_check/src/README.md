## マクロの主要機能

### 1. フォーム必須入力チェック（ValidateCheck）
- **対象シート**: 机上チェックシート、現地調査
- **動作**: 必須入力項目の未入力チェック → 未入力セルを黄色背景で強調、ダイアログ表示
- **詳細**: シート保護を一時解除 → チェック実行 → シート再保護（図形操作許可）
- **チェック種別**:
  - `ValidateDesktopSimple` - 単純必須チェック（指定セルが空でないか）
  - `ValidateDesktopGroupsA/B` - ラジオグループチェック（いずれか選択必須）
  - `ValidateDesktopConditional` - 条件付き必須チェック
  - `ValidateOnsiteBaseChecks` - 現地調査基本チェック
  - `ValidateOnsiteSpecialPatterns1-8` - 現地調査特殊パターンチェック（8種類）
  - `ValidateB2CHighlightedCells` - B2/C判定セルの入力チェック

### 2. データクリア処理（ClearAllInputData）
- **対象シート**: 机上チェックシート、現地調査
- **動作**: 入力データを全クリア（確認ダイアログ付き、元に戻せない）
- **詳細**: セル値クリア + 背景色リセット

### 3. B2C判定処理（ExecuteB2CJudgment）
- **対象シート**: 現地調査 → 評価b2劣化事象/評価c劣化事象 に転記
- **動作**: 現地調査シートの■マーク（b2/c判定）を自動収集し、各劣化事象シートにデータ転記
- **詳細**: 場所別（敷地及び地盤、各点検口内、建築物外部、建築物内部、その他）のセル定義に基づき判定

### 4. 画像挿入機能（シートモジュール共通）
- **対象シート**: 定形写真、評価c劣化事象、評価b2劣化事象、評価c類似
- **動作**: セルダブルクリック → ファイル選択ダイアログ → 画像をセルサイズに合わせて挿入
- **バリエーション**:
  - Sheet2（定形写真）: 16箇所の固定アドレス
  - Sheet3/Sheet4: B列/L列の動的パターン（行番号を規則的に生成）
  - Sheet8: 改良版（結合セル対応、より安定したセル判定）

### 5. ユーティリティ関数
- `ValidateSimple` - 単一セル必須チェック
- `ValidateGroup` - ラジオボタングループチェック
- `ValidateGroupOr` - 複数グループのOR条件チェック
- `ValidateCondition` - 条件付き必須チェック
- `SetCellBackgroundColor` - セル背景色設定
- `HexToRgb` - HEXカラーコードからRGB変換
- `IsCellEmpty` - セル空チェック
- `IsCheckboxCell` / `IsKnownCheckboxCell` - チェックボックスセル判定
- `IsSpecialNothingCell` - 特殊「該当なし」セル判定
