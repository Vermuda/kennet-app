現地調査チェックシート UIモック

実際の検査項目書（101項目・9カテゴリ）を基にした5つのUIパターンです。
LocalStorageで入力データを共有しているので、パターン間でデータが引き継がれます。

No.1-7（7項目）地盤・敷地・擁壁など
外部② 各点検口内
No.8-29（22項目）床下・小屋裏など
外部③ 基礎
No.30-39（10項目）コンクリート・モルタルなど
外部③ 外壁
No.40-57（18項目）ひび割れ・シーリングなど
外部③ 廊下・階段等
No.58-74（17項目）手摺・排水など
外部④ 屋根及び屋上
No.75-91（17項目）屋根材・防水など
外部⑤ 管理状況
No.92-94（3項目）清掃・管理など
鉄筋・シュミットハンマー
No.95-96（2項目）鉄筋確認・強度など
その他 違法性関係
No.97-101（5項目）増築・用途など

✨ 共通機能

✓
複数評価：1つの項目に対して複数の評価を追加可能（例：複数の水抜きパイプ）
✓
場所メモ：各評価に場所メモ（例：北側、2階など）を記録可能
✓
進捗表示：リアルタイムで完了数・進捗率を表示
✓
データ永続化：LocalStorageに自動保存（パターン間で共有）
✓
評価レベル：a（良好）/ b1（軽度）/ b2（中度）/ c（重度）/ −（対象外）
✓
カテゴリジャンプ：100項目以上でも大項目レベルで素早く移動
🎯

🧭 方位設定UIモック（5パターン）

スマートフォンでの操作性を重視した方位設定UIの提案です。

📱

パターン1: ボトムシート型

図面を最大化表示し、下部シートで操作。タブで回転/移動/サイズを切り替え。

✓ 図面表示エリア最大 ✓ 直感的なタブ切替
🎛️

パターン2: 円形ダイヤル型

下部に大きな円形ダイヤルを配置。ドラッグ操作で直感的に角度調整。

✓ ダイヤル操作が直感的 ✓ 細かい角度調整可能
👆

パターン3: ジェスチャー特化型

UI最小化。1本指で回転、2本指で移動、ピンチでサイズ変更。

✓ 画面全体を使用 ✓ ジェスチャーで全操作
🎚️

パターン4: スライダー集約型

角度・X位置・Y位置・サイズを全てスライダーで調整。明確なコントロール。

✓ 操作が明確 ✓ 数値での確認が容易
🔘

パターン5: フローティング型

図面全画面表示。四隅にフローティングボタン配置。8方位プリセット付き。

✓ 図面が見やすい ✓ 直接ドラッグ操作
💡

推奨パターン

スマホ：パターン1 or 5
iPad：パターン3（ジェスチャー）
